# 总论
## 递进式的思维记录
### 以百度网盘为例，我们来想想程序到底是怎么帮助我们的以及程序是怎么起作用的
* 我喜欢精美的原画和酷炫的动画，有时候当我看见这些东西的时候我就会把他们保存起来，以后想看的时候就看，可是这个时候就出了问题
* 将东西保存到硬盘、U盘的时候，我需要一个接口才能打开U盘和硬盘来查看这些图片，可是我想只要拿出手机就可以随时随地的看到，于是
* 我需要一个网络存储的产品，这个产品可以让我存储东西，也可以让我直接就可以查看我存储的东西。我通过浏览器锁定了目标为网盘。
* 将目标锁定了网盘之后，我便将它下载下来。
* 下载下来之后，我双击了网盘.exe开始启动程序
* 程序启动之后，程序通知操作系统，操作系统开启进程运行程序，同时操作系统准备程序运行的所有前置条件
* 操作系统将程序从硬盘放到内存中去，同时告知CPU开始去内存地址中找寻程序并执行程序
* 当开始执行程序的时候就到了程序的层面 程序通过编译器被编译为native code(机器代码)被执行
* 程序被执行的时候，就是操作系统不断切换指令，内存不断开辟空间，操作系统调用别的硬件软件一起运行
